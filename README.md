# Movie Recommendation System - Content Based Filtering

## Overview
This project implements a Content Based Filtering recommendation system 
on the MovieLens dataset from Kaggle. As Group 1, the assigned task was 
to build a content based filtering system by extracting movie features 
and building user profiles from historical preferences.

## Group
Group 1 - Bora Malaj
Assigned Task: Content Based Filtering

## Dataset
- Source: Kaggle - MovieLens Dataset
- Samples: 9742 movies
- Features: movieId, title, genres
- Task: Content Based Movie Recommendation

## Steps Covered
1. Data Loading and Basic Information
2. Feature Extraction - genres, year from title
3. One Hot Encoding of genres
4. Movie Feature Matrix construction
5. Cosine Similarity computation
6. Movie Recommendation Function
7. User Profile building from historical preferences
8. Personalized recommendations per user
9. Genre Distribution visualization

## Libraries Used
- pandas, numpy, matplotlib, seaborn, scikit-learn
